About Weather Wear

Weather Wear is your intelligent clothing recommendation app that helps you dress appropriately for any weather condition.

Features:

  • Smart Recommendations: Get personalized clothing suggestions based on temperature and weather conditions
  • Real-time Weather: Access current weather data from OpenWeatherMap API
  • Location Services: Use your current location for instant weather updates
  • City Search: Check weather for any city worldwide
  • Dynamic Background: Beautiful animated backgrounds that change with weather conditions

How it works:

Simply enter a city name or use your current location, and Weather Wear will provide you with:

  • Current temperature
  • Weather conditions
  • Personalized clothing recommendations

Technology:

Built with HTML5, CSS3, and JavaScript, featuring:

  • Vanta.js for beautiful animated backgrounds
  • OpenWeatherMap API for accurate weather data
  • Responsive design for all devices
  • Geolocation services
